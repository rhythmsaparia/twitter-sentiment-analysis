# Twitter Sentiment Analysis

**Project Description**

* This project aims to do sentiment analysis on twitter tweets during pandemic. Machine learning and Deep learning techniques used : Naive bayes, Transformer(BERT).

**Technologies Used**

* Key Technologies:
    * Transformers (Bert)
    * TensorFlow/Keras
    * Twitter API
    * matplotlib, seaborn
    * sklearn

* **Python Version:**
* 3.10.x

**Dataset**
Custom dataset created using twitter API

**Results**

* Naive Bayes:
  * Accuracy: 0.70
  * Precision: 0.70
  * Recall: 0.70
  * F1: 0.70

* Bert:
  * Precision: 0.88
  * Recall: 0.88
  * F1 Score: 0.88
